module module¶

class module.Module[source]¶

Bases: object

The representation of a Cortix module. This class is to be inherited by every Cortix module. It provides facilities for creating modules within the Cortix network.

connect(port_name, connected_port)[source]¶

A simpler interface to create module connectivity. Connect the module port with port_name to a given connected_port.

get_port(name)[source]¶

Get port by name; if it does not exist, create one.
