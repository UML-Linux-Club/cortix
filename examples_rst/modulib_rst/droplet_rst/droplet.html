_Droplet__evolve(cortix_time=0.0, cortix_time_step=0.0)¶

ODE IVP problem: Given the initial data at t=0, u_1(0) = x_0, u_2(0) = v_0 = \dot{u}_1(0) solve \frac{\text{d}u}{\text{d}t} = f(u) in the interval 0\le t \le t_f. When u_1(t) is negative, bounce the droplet to a random height between 0 and 1.2\,x_0 with no velocity, and continue the time integration until t \le t_f.
